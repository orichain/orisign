# Security Policy

## Supported Versions
This repository is a research prototype. No versions are officially supported
for production use.

## Reporting a Vulnerability
If you discover a security issue, please report it privately by opening a
GitHub Security Advisory or contacting the maintainers directly. Please do not
open public issues for security vulnerabilities.

## Scope
This project is intended for educational and experimental purposes only and
has not undergone formal cryptographic review or security auditing.
